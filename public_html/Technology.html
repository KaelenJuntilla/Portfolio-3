Technologies Page

HTML(Hypertext Markup Language)

Hypertext markup language or Html is used for designing online websites and also configure them.It is used to import codes or text that will be seen on the website. you can put pictures,videos and many other things using html.It is used for putting text in the body and header.

CSS(Cascading Style Sheet)

Casacading style sheet or Css is used for designing the website with colors,pictures and many more.Mostly every web designer uses this style designer on their website.This is the reason why you see all these websites with a bunch of creativity.Css is one of the most powerful tools to use because it is used to create and style a web page.

Bootstrap(Web Designer's Tool)

Bootstrap is a web designers tool to create websites using a grid system that just uses rows and colummns to design websites.It is an easier tool to use than your Html or css.Html is just the basics on how to code.once you learn how to code you use bootstrap to create them cool looking websites you see today.

Javascript

Javascript is popular designers tool for making websites.It is often used to create simple games.The coding in the javascript is for games or websites that use keyboard functions or click functions. There are specific coding to perform certain tasks like go up with a key or clicking a restart button, or even perform the same thing over and over again.Javascript code can be easily pasted from the website to your required place in your code.

Jquery uses the main designers tools but just makes it easier to do more things with one line of code.The main designers tools are Html,Css, and Javascript.Its better to understand these main designers tools because using Jquery makes it easier but isn't something new to learn.I could create a simple button that will hide the text and make it reappear with one line of code.
